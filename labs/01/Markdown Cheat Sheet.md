## Markdown Cheat Sheet

### **Basic Formatations**

## **Headings:**

To make headers in Markdown, you preface the phrase with a hash mark (#). 
You place the same number of hash marks as the size of the header you want.

    # Header one = 
Preview: 
# Header one

    ## Header two
Preview: 
## Header two

    ### Header two
Preview: 
### Header two


## **Italics:**

By using a (*) or (_) around your text makes it slanted to emphasize it.


    *This is italic* 
  
Preview: *This is italic*

    _This is also Italic_

Preview: _This is also Italic_

## **Bold:**

By using double asterisks ** or double underscore __ around your text makes the text thicker and darker to stand out.

    ** This text is bold **

Preview: **This text is bold**

     __ This text is also bold __

Preview: __This text is also bold__


## **Strikethrough:**

By using ~ before and the end of a word or Phrase it draws a line through the text.

    ~Text~

Preview: ~Text~


## **Inline Code:**

By using backticks ` around text its Displays text in a code-style font.

    Use the `print()` function.

Preview: Use the `print()` function.


## **Lists**


## **Unordered Lists:**

 to make a unordered List use Use -, *, or +.

     - Student 1
     - Student 2
     - Student 3

Preview:

- Student 1
- Student 2
- Student 3

## **Ordered Lists:**

  to make ordered List use numbers followed by a period.

      1. First
      2. Second
      3. Third

Preview:

1. First
2. Second
3. Third

## **Nested Lists:**

Indent items with two spaces or a tab.

      1. Main item
         - Sub-item
         - Another sub-item

Preview:

1. Main item
   - Sub-item
   - Another sub-item


## **Links & Images**


## **Inline Links:**
by inserting a Link to specific website use [text] (URL)

        [Visit GitHub](https://github.com)

Preview:

[Visit GitHub](https://github.com)


## **Reference-style Links:**

By placing the URL as a reference and referring to it in a link, it can be sure any additional links referring to it get the same URL, even by changing the URL.

        [GitHub][1]

        [1]: https://github.com

Preview:  

[GitHub][1]

[1]: https://github.com
